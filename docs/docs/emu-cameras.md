# EMU Camera Serials

List of supported EMU camera serial numbers:

| Camera Serial | Position | Bit Number | 
|--------------|----------|------------ |
| 18486634     | F1       |8            |
| 23512908     | F2       |9            |
| 18486644     | F3       |10           |
| 18486638     | B1       |11           |
| 23512014     | B2       |13           |
| 23512906     | R1       |15           |
| 23512012     | R2       |14           |
| 23505577     | R3       |12           |

Use these serials when mapping cameras for video synchronization.
